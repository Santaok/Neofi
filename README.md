# Neofi
//////////////////////////////////////////////////////

# О проекте
NeoFI — это веб-приложение для управления цифровыми активами в сфере децентрализованных финансов (DeFi). Оно позволяет пользователям создавать диверсифицированные портфели криптовалют в один клик, используя готовые стратегии, основанные на популярных криптоиндексах и трендах.

Ключевая задача
Основная проблема, которую решает NeoFI, — сложность входа в мир децентрализованных финансов для обычных пользователей. Традиционные DeFi-платформы требуют глубоких технических знаний и ручного управления активами. NeoFI автоматизирует этот процесс, предлагая простой и интуитивно понятный интерфейс для создания и управления сбалансированным криптопортфелем.

Кто пользователь
Целевая аудитория NeoFI — это как новички в криптовалютах, которые хотят инвестировать без сложных настроек, так и опытные трейдеры, стремящиеся автоматизировать управление своими активами и диверсифицировать риски.

Технология
Проект построен на блокчейне BNB Smart Chain, а нативный токен NEOFI используется как средство обмена в экосистеме, а также для стимулирования участия и вознаграждений. Общий объём эмиссии токена составляет 350 миллионов монет. Платформа привлекла инвестиции от ряда венчурных фондов.
